description: Parça branch'inde küçük checkpoint commit
---

Checkpoint commit at:

1. Branch kontrolü: main'de İSEN DUR (main-guard zaten bloklar) — parça branch'ine geç.
2. `git status` + değişen dosyaları gözden geçir; alakasız/beklenmedik dosya varsa commit'leme, bildir.
3. Anlamlı küçük commit: `checkpoint(<parça>): <ne yapıldı>`.
4. Tek satır durum raporu: ne bitti, sırada ne var.
